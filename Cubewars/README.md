# Cube Wars :gun:

This is my attempt of creating the Boxhead 2D zombie game.

## :mag_right: Installation

```bash
# clone the project
git clone https://github.com/ian13456/cube-wars.git

# used to run server locally
yarn global add serve

cd cube-wars

# run the website locally
serve
```

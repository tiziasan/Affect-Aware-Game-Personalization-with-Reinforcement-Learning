

This is the supplemental material for the paper "Affect-Aware Game Personalization with Reinforcement Learning: How to Improve Players' Engagement, Performance, and Emotions"
- The folder Cubewars contains the code for Cubewars game with the relative instructions to install it
- The folder MFRL contains the code for the Model-Free Reainforcement Learning algorithm, and the server to comunicate with the game, with the installation instruction
- The folder data contains the post-experimentation survey used to reply to Research Question 5 and the raw data of all the 36 partecipant experiments
